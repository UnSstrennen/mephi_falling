# Программа для расчёта свободного падения тела

## Функциональные возможности

* Рассчитывает время падения тела с определённой высоты **h** в зависимости от заданного ускорения свободного падения **g** (Величина **g** по умолчанию задана для планеты Земля).
* Рассчитывает расстояние от тела до поверхности планеты в указанное время **t**.
* Рассчитывает время, за которое тело достигнет определённой высоты **h**.
* Строит график **h(t)**, где **h** - расстояние, пройденное телом за время падения, **t** - время падения тела.

## Описание математической модели падающего тела

Падение тела как физическое явление есть равнопеременное движение(РПД) под действием силы тяжести **g**. Сила тяжести является определённой постоянной величиной для планет Солнечной Системы.
Свободное падение является частным случаем РПД:
* При свбодном падении тело не обладает начальной скоростью (начальная скорость равна нулю).
* На тело не воздействуют или пренебрижимо малы внешние силы.

Время, за которое тело совершит свободное падение на поверхность планеты, определяется по формуле
![img](http://www.sciweavers.org/tex2img.php?eq=t%3D%20%5Csqrt%5B%5D%7B%20%5Cfrac%7B2h%7D%7Bg%7D%7D%20&bc=White&fc=Black&im=png&fs=12&ff=arev&edit=0)
, где **h** - высота, с которой брошено тело, **g** - ускорение свободного падения. Используя данную формулу, также можно узнать время, за которое тело пройдёт определённое расстояние. В таком случае **h** - расстояние, которое необходимо преодолеть телу.

Для расчёта расстояния, пройденного телом за время **t**, применяется обратная формула - ![img](http://www.sciweavers.org/tex2img.php?eq=h%3D%20%5Cfrac%7Bgt%5E%7B2%7D%7D%7B2%7D%20&bc=White&fc=Black&im=png&fs=12&ff=arev&edit=0)
, где **g** - ускорение свободного падения, **t** - время, за которое тело должно должно преодолеть заданное расстояние **h**.

Рассчитав расстояние, пройденное телом за время **t**, также можно узнать расстояние тела От поверхности планеты в указанный момент времени **t** по формуле 
![img](http://www.sciweavers.org/tex2img.php?eq=h%20%3D%20H%20-%20%5Cfrac%7Bgt%5E%7B2%7D%7D%7B2%7D%20&bc=White&fc=Black&im=png&fs=12&ff=arev&edit=0)
, где H - высота начала падения тела от поверхности планеты, **g** - ускорение свободного падения, **t** - время, за которое тело должно должно преодолеть заданное расстояние **h**.
